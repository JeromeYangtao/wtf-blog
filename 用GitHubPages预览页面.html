GitHub中有一个GitHub Pages功能，能让我们方便的预览HTML文件。 具体步骤如下： ##### 1.登陆GitHub,新建repo ![](http://upload-images.jianshu.io/upload_images/3474707-79ce8a10fe867373.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240) ![](http://upload-images.jianshu.io/upload_images/3474707-f16cdb76415c4ac1.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240) ##### 2.进入【setting】页面 ![](http://upload-images.jianshu.io/upload_images/3474707-e0576a4a765db161.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240) ##### 3.往下滚，开启 GitHub Pages 功能，得到一个「预览地址」，我的预览地址是[https://jessmremero.github.io/demos/](https://jessmremero.github.io/demos/) ![](http://upload-images.jianshu.io/upload_images/3474707-8a052f6fbace1abd.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240) ###### 4.在[预览地址]后面加上test.html等HTML文件即可预览 比如你的 html 路径是 test/test.html，那么预览链接就是[https://jessmremero.github.io/demos/test/test.html](https://jessmremero.github.io/demos/test/test.html) ![](http://upload-images.jianshu.io/upload_images/3474707-85f3484e5819ad5f.png?imageMogr2/auto-orient/strip%7CimageView2/2/w/1240) ##### 5、在浏览器输入 [https://jessmremero.github.io/demos/test/test.html](https://jessmremero.github.io/demos/test/test.html) 这样，你就能自己随时预览 GitHub 里的页面了！
